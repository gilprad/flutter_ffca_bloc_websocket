# Trading View WS

Flutter project untuk eksperimen/implementasi **TradingView** dengan koneksi **WebSocket (WS)**.

## Prasyarat

- **FVM (Flutter Version Manager)** terpasang
- **Flutter**: 3.38.9
- **Dart**: 3.10.8
- **JDK**: 17
- **Android Build Tools**: 36.1.0

## Menjalankan (disarankan via FVM)

> Pastikan menjalankan project memakai versi Flutter yang sesuai.

1. Install Flutter versi yang dibutuhkan (sekali saja):

```bash
fvm install 3.38.9
```

2. Set versi Flutter untuk project ini:

```bash
fvm use 3.38.9
```

3. Ambil dependencies:

```bash
fvm flutter pub get
```

4. Jalankan aplikasi:

```bash
fvm flutter run
```
